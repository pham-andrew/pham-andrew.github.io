## Services

I can build your app; however, I will eventually have to hand it off to you or your team. Everything requires maintenance, changes need to be made, security needs to be updated, etc. There is no such thing as an active website without maintenance. 

### What I can do
1. Deploy a full stack react application to the web for use as a prototype or proof of concept.
2. Deploy a full stack react application and pass it off to your team to maintain.
3. Launch a static front-end only page and transfer its hosting/domain/ownership to you.

### What I can't do
1. Deploy, maintain, and update an application indefinitely. I'm employed full time and it wouldn't be fair to you if something came up and your project was on the backburner.
2. Attempt to make major changes to an already existing application especially with stacks that are not PERN or LAMP. Any emergency fixes I duct tape together for a temporary fix will likely result in other unfavorable behavior.
